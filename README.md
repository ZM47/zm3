这是我用django架构做的一个网站，内容用的是scrapy框架爬取豆瓣读书的书签图书，用mysql关系型数据库进行存储。

前端页面大体使用bootstrap框架，自己的ccs和js修改而成。分页设置和搜索设置，登录，注册都是用django自带的框架，
